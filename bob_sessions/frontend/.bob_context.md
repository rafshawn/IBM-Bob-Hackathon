## Architecture Philosophy

### Islands Architecture
```
┌─────────────────────────────────────────┐
│         Static HTML (Astro)             │  ← Zero JavaScript
│  ┌─────────┐  ┌─────────┐  ┌─────────┐ │
│  │ Header  │  │ Content │  │ Footer  │ │
│  └─────────┘  └─────────┘  └─────────┘ │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │   Vue Island (Interactive)      │   │  ← JavaScript only here
│  │   - Scan Form                   │   │
│  │   - Progress Tracker            │   │
│  │   - Issue Table                 │   │
│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

**Why This Matters**:
- Public service workers often have older hardware
- Low bandwidth networks are common
- Accessibility is critical
- Performance = Accessibility

## API Integration Pattern

### 1. Create API Client
```typescript
// src/lib/api.ts
const API_BASE = import.meta.env.PUBLIC_API_URL || 'http://localhost:8000/api/v1';

export async function startScan(url: string) {
  const response = await fetch(`${API_BASE}/scans`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ url })
  });
  
  if (!response.ok) {
    throw new Error('Failed to start scan');
  }
  
  return response.json();
}
```

### 2. Use in Vue Component
```vue
<script setup lang="ts">
import { ref } from 'vue';
import { startScan } from '../lib/api';

const url = ref('');
const loading = ref(false);

async function handleSubmit() {
  loading.value = true;
  try {
    const result = await startScan(url.value);
    // Handle success
  } catch (error) {
    // Handle error
  } finally {
    loading.value = false;
  }
}
</script>
```

### 3. Polling Pattern
```typescript
// Poll for scan status
let pollInterval: number;

onMounted(() => {
```

## Error Handling

### API Errors
```typescript
async function fetchData() {
  try {
    const response = await fetch('/api/data');
    
    if (!response.ok) {
      throw new Error(`HTTP ${response.status}: ${response.statusText}`);
    }
    
    return await response.json();
  } catch (error) {
    console.error('Failed to fetch data:', error);
    // Show user-friendly error message
    return null;
  }
}
```

### User Feedback

## Common Patterns

### Loading State
```vue
<script setup>
const loading = ref(false);
const data = ref(null);

async function loadData() {
  loading.value = true;
  try {
    data.value = await fetchData();
  } finally {
    loading.value = false;
  }
}
</script>

<template>
  <div v-if="loading">Loading...</div>
  <div v-else-if="data">{{ data }}</div>
  <div v-else>No data</div>
</template>
```

### Form Validation
```vue
<script setup>
const url = ref('');
const errors = ref<string[]>([]);

function validateUrl() {
  errors.value = [];
  
  if (!url.value) {
    errors.value.push('URL is required');
  } else if (!url.value.startsWith('http')) {
    errors.value.push('URL must start with http:// or https://');
  }
  
  return errors.value.length === 0;
}

function handleSubmit() {
  if (validateUrl()) {
    // Submit form
  }
}
</script>
```
